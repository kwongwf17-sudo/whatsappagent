# Blackhead Remover Image Assets

Save the attached images in this folder using these filenames:

- `product-1.png` - See The Difference image.
- `product-2.png` - 4 Key Functions image.
- `product-3.png` - Built With Smart Features image.
- `testimonial-1.png` - Testimony 1 image.
- `testimonial-2.png` - Testimony 2 image.
- `testimonial-3.png` - Testimony 3 image.
- `testimonial-4.png` - Testimony 4 image.
- `price.jpeg` - Blackhead Remover price image.

In local demo mode, the server serves these files from:

```text
http://localhost:3000/assets/blackhead-remover/FILENAME
```

For real WhatsApp sending, expose the server with HTTPS and set `PUBLIC_BASE_URL`, or replace the catalog image URLs with hosted HTTPS links.
